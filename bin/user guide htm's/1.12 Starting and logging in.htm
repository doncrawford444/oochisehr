1.12 Starting and logging in

To start Oochis EHR, double-click Oochis EHR's desktop icon (Oochis EHR icon). The login screen (image of login) will be displayed.

Enter a user name and password at the login screen. There is a limited number of login attempts allowed before the system administrators is required to login. The number of login attempts allowed is set by the administrator in 'System' → 'Set-up'. There is also a time limit of 2 minutes allowed between keystrokes before the login screen clears itself. The workspace (image of workspace) will appear when log in is successful.

To logout click on the 'Login' button to the left of the task bar (image of 'Login' button in the workspace). The login screen is always displayed when not logged in.

To exit Oochis EHR, click on the 'Login' button, then, 'Exit.'

Oochis EHR keeps running in the background when exited. This is to process incoming and outgoing requests from other systems. Exiting Oochis EHR at any point in the work flow does not interfere with any processing or change any existing information.
