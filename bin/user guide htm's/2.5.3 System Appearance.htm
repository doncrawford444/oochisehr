2.5.3 Appearance

'Appearance' ('System' → 'Appearance'):
Oochis EHR’s appearance can be changed quite dramatically. Users authorized to manage their 'Appearance', are able to change the wallpaper, system font, window background color and transparency, and border style. Users with administrator privileges set the default appearance for all users they administer.

'Window background' ('System' → 'Appearance' → 'Window background'):
Change the window background color and transparency.

'Window background color' ('System' → 'Appearance' → 'Window background' → 'Window background color'):
Click on a swatch to change the background color of windows. The chosen background color is still affected by the translucency.

'Window background translucency' ('System' → 'Appearance' → 'Window background' → 'Window background translucency'):
Adjust the slider to change the translucency of window backgrounds.

'Wallpaper' ('System' → 'Appearance' → 'Wallpaper'):
To change the wallpaper, click (or double-click) on a jpeg file in the file-navigation box. Only folders and jpeg files are displayed.

The wallpaper may be fitted to the screen using the 'center', 'stretch', and 'zoom' commands (buttons) as described below:

'Zoom' ('System' → 'Appearance' → 'Wallpaper' → 'Zoom'):
Clicking, 'Zoom', zooms the image in or out to fit the screen. Some of the image's area on top and bottom, or the sides may not be displayed if the image's height-to-width ratio is different than the screen's.

'Center' ('System' → 'Appearance' → 'Wallpaper' → 'Center'):
To 'Center' the wallpaper image means the image will not be re-sized; the image will simply be placed at the center of the screen against a white backdrop. Some of the sides of larger image's may not be visible.

'Stretch' ('System' → 'Appearance' → 'Wallpaper' → 'Stretch'):
To 'Stretch' the wallpaper image means the image’s corners will become the corners of the screen. The image may be distorted accordingly.

'Remove' ('System' → 'Appearance' → 'Wallpaper' → 'Remove'):
Removes the wallpaper, leaving a white desktop background.

'Border-theme' ('System' → 'Appearance' → 'Border-theme'):
There are four border-themes: 'Slick'; 'Slate'; 'Rounded'; and 'High contrast' (see figure). 'Slick', 'Slate', and 'Rounded' change borders of visible objects. High contrast is an accessibility feature that maximizes visibility.

'Slick' ('System' → 'Appearance' → 'Theme' → 'Slick'):
Changes the border-theme to a slick border. The borders do not raise or lower their objects, giving a slick, or nestled, appearance. This is ideal for touch screens, as it correctly renders object depths to the same 2-dimensional screen surface.

'Slate' ('System' → 'Appearance' → 'Theme' → 'Slate'):
Changes the border-theme to a raised, step-like border. The borders raise their objects.

'Rounded' ('System' → 'Appearance' → 'Theme' → 'Rounded'):
Changes the border-theme to a rounded border. The rounded border edges make their objects appear to be raised up from their surroundings.

'High contrast' ('System' → 'Appearance' → 'Theme' → 'High contrast'):
High contrast is an accessibility feature that maximizes the visibility of objects using thick, black borders. Also, in order to maximize textual contrast, the controls' (i.e., buttons') backgrounds are changed to white.

'Font' ('System' → 'Appearance' → 'Font'):
To change the font type, click on 'System' → 'Appearance' → 'Font'. A font type of 'Ariel', 'Serif', or 'Electron' is chosen by selecting the radio button to the right of the font type (see figure). Font size can be adjusted from 10 to 22 point sizes by entering the desired size in the indicated field.

'Enter font size (from 12 to 22): ' ('System' → 'Appearance' → 'Font' → 'Ariel'):
Enter a number from 12 to 22 changes the system font.

'Ariel' ('System' → 'Appearance' → 'Font' → 'Ariel'):
Clicking this radio-button changes the system font to 'Ariel'.

'Serif' ('System' → 'Appearance' → 'Font' → 'Serif'):
Clicking this radio-button changes the system font to 'Serif'.

'Electron' ('System' → 'Appearance' → 'Font' → 'Electron'):
Clicking this radio-button changes the system font to 'Electron'.
