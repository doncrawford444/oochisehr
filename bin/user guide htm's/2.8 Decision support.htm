2.8 Decision support

The Decision support (window) contains basic biomedical tools, a new yet dogmatic translational medicine functionality, and also provides 'Treatment analyses' ('Rules', 'Statistics', 'Cost analysis'; 'Treatment effect'; 'Decision models'; and, 'Life expectancy') and 'Statistics' tools (functions). Decision Support also provides a query feature, and an encapsulated Internet browser with safe, tunneled, certified, connections to links to on-line protocol and patient guidelines and services.

The Decision Support window refreshes (re-calculates) it's informational content to pertain to the single, open chart only. When a chart is closed, features that use chart data ('Decision Support's 'Translational Medicine', and 'Treatment Analysis's life-curves, ) no longer display information...
