2.9.2 Network files

'Network files' ('Practice Assistant' → 'Network files'):
For syncing network files for billing, claims, disease reporting. (Also see '-Networking' on pg 56).

'Export network directory' ('Practice Assistant' → 'Network files' → 'Export network directory'):
'Export network directory' is a directory of shared health data files (.CCR, .CDA, and .XML with HL7 messages as content) meant to be accessed by other programs, such as billing and claims, practice management, and other EHR solution softwares. The panel below displays the names of all network files in the 'Export network directory'.

Enter a network directory accessible by other programs. The purpose of the Network output directory is to read health files in standard formats. The 'Export network directory' could be (and likely is) the same as the input network directory.

Other types of integration involves, for example, installing real-time data streams through object brokers rather than read and writes to network files.

All data imported are treated as confidential patient-and-provider information.

'Export file name: ' ('Practice Assistant' → 'Network files' → 'Export file name: '
Name the export file to be created. Export network files are files meant to be accessed by other programs such as billing and claims, and practice management. Exportable files are displayed in the top-most (large) panel.

'Browse' ('Practice Assistant' → 'Network files' → 'Browse'):
Browse file system for export directories.

'Export (create file)' ('Practice Assistant' → 'Network files' → 'Export (create file)'):
Click 'Create an export file' and the new, named network file will be displayed in the panel below. Click on any network file in the panel below to display it's contents (in the same panel).

'Import network directory' ('Practice Assistant' → 'Network files' → 'Import network directory'):
'Import network directory' is a directory of shared health data files (.CCR, .CDA, and .XML with HL7 messages as content) meant to be accessed by other programs, such as billing and claims, practice management, and other EHR solution softwares. The panel below displays the names of all network files in the 'Import network directory'.

Enter a network directory, accessible by other programs, and perhaps even specified by other programs. The purpose of the Network output directory is to read health files in standard formats. The 'Import network directory' could be (and likely is) the same as the export network directory.

Other types of integration involves, for example, installing real-time data streams through object brokers rather than read and writes to network files.

All data imported are treated as confidential patient-and-provider information.

'Import file name: ' ('Practice Assistant' → 'Network files' → 'Import file name: '
Import network files are files meant to be accessed by Oochis software, such as from patients, other EHR solftware, billing and claims, and practice management. The names of imported files are displayed in the bottom-most (large) panel.

'Browse' ('Practice Assistant' → 'Network files' → 'Browse'):
Browse file system for import directories.

'Import' ('Practice Assistant' → 'Network files' → 'Import'):
Click 'Import' and the new, named network file will be displayed in the panel below. Click on any network file in the panel below to display it's contents (in the same panel).
