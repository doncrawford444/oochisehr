2.6.1 Menu bar

'Menu bar' ('Charting' → 'Menu bar'):
The 'Charting' 'Menu bar' is at the top of the window under the title bar. The menu bar contains seven buttons:

  • 'New chart': Creates a new chart.
  • 'Open chart': Opens an existing chart.
  • 'Export chart': Exports EHR data into various formats.
  • 'Print': Prints chart pages.
  • 'Sign off and process': Signs off and process the entries
  • 'Close chart': Closes the chart.
  • 'Note pad': Opens a note pad.
  • 'Summary and discharge': Opens a note pad with summary and discharge information.
  • 'Find': Find feature.

'New chart' ('Charting' [menu bar] → 'New chart'):
To create a new chart, click 'New chart', and choose a chart template in the drop down menu. Chart templates are created in 'System' → 'Templates'. The process of creating a chart template is similar to creating a document in a word processor. See 'Templates' for information on how to create chart templates. Enter, "example," in the ‘New chart’ drop-down menu to get a feel for what a new (but empty) chart looks like.

'Open chart' ('Charting' [menu bar] → 'Open chart'):
Click, 'Open chart' to open a chart. A dialog will be displayed for entering patient identifiers. Unique patient identifiers are: chart ID; social security number; and driver's license number. Charts can also be looked up using patient names and birth dates. Patient names and birth dates are not unique and may return multiple charts. If multiple charts are returned, demographic information in each chart is displayed, enabling further selection.

To open a recently opened chart, select (click on) a recently opened chart from the list at the bottom of the 'Open chart' dialog box. Recently opened charts are identified with the chart's chart ID, and the patient's name, sex, and birth date.

'Close chart' ('Charting' [menu bar] → 'Close chart'):
Click, 'Close' to close a chart. If any chart additions have not been signed off, a dialog panel is displayed prompting the user to 'Sign-off and close (the) chart', or to 'Close (the) chart without signing off'.

'Find' ('Charting' → 'Menu bar' → 'Find'):
Click 'Find' to search a chart for specific content. A search dialog will be displayed to enter search queries, constrain searches to sections or pages, and navigate through returns.

'Summary and discharge' ('Charting' [menu bar] → 'Summary and discharge'):
Clicking 'Summary and discharge' opens a note with 'Summary and discharge' information filled-in automatically using chart information and rules-based processing (such as supplying recommended handouts automatically when specific encoded items are ordered and via {{{those created using scripts and recorded user events}}} ).

'Note pad' ('Charting' [menu bar] → 'Note pad'):
Narrative (any and all narrative, although especially pertaining to the note pad notes) is not encoded nor connected to an EHR unless explicitly added to a chart. Notes can be added to any chart item by activating the 'Attach to current order' button, or by dropping or pasting the note (or text content) onto the item. Notes can also be added directly as encoded narratives by dropping or pasting the note (or text content) on CPOE panel (where it is processed as a coded entry) of an open chart.

Attach a clinical note to a pending order by right-clicking the order slip in 'Activities' CPOE panel. The note pad will open, and the note attached as narrative to the order. Attached notes are displayed in the CPOE's order slip panel. Closing the note pad with unsaved changes prompts the user to save any changes before closing.

Content can be dropped or pasted onto the workspace from another application or directory. The workspace serves as a desktop-like area to store working items.

Content can be exchanged between Oochis EHR's note pad and third party applications by cutting, copying, and pasting, as well as dragging and dropping.

Remember, you may click CTRL + ALT + M to minimize Oochis EHR to your operating system’s task bar in order to cut/copy and paste and drag and drop between Oochis EHR and other applications.

'New' ('Charting' → 'Menu bar' → 'Note pad' → 'New'):

'Close' ('Charting' → 'Menu bar' → 'Note pad' → 'Close'):
Closes the note. If the note is not saved, a prompt will be displayed requesting whether to save any changes before closing.

'Open' ('Charting' → 'Menu bar' → 'Note pad' → 'Open'):
Displays a directory tree and all of all standard document file formats (such as rtf, doc, html, txt, pdf, odf) in the current directory.

'Save' ('Charting' → 'Menu bar' → 'Note pad' → 'Save'):
Clicking 'Save' saves the note.

'Save as' ('Charting' → 'Menu bar' → 'Note pad' → 'Save as'):
Clicking 'Save as' saves the note under a different name.

'Undo' ('Charting' → 'Menu bar' → 'Note pad' → 'Undo'):
Clicking 'Undo' reverses the last typed entry.

'Redo' ('Charting' → 'Menu bar' → 'Note pad' → 'Redo'):
Clicking 'Redo' reverses the last 'Undo'.

'Print' ('Charting' → 'Menu bar' → 'Note pad' → 'Print'):
Opens a print dialog panel for printing.

'Send as note' ('Charting' → 'Menu bar' → 'Note pad' → 'Send as note'):
Opens the Messaging window and attaches the user index to a new email.

'Attach to current order' ('Charting' → 'Menu bar' → 'Note pad' → 'Attach to current order'):
Files and content on the Note pad are not encoded nor connected to an EHR unless explicitly added as an item or to an item (as an attachment). Clicking 'Attach to current order' adds the note to a chart item (the note becomes EHR content). Alternatively, you may drop or paste the note (or text content) onto the chart item. Notes can also be added directly as encoded narratives by dropping the note pad or pasting the note's content onto the CPOE panel, for further processing as an encoded chart entry.

'Find' ('Charting' → 'Menu bar' → 'Note pad' → 'Find'):
The 'Note pad's 'Find' feature. Clicking 'Find' displays a search panel for entering queries and navigating through returns.
